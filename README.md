## meteor-sendgrid - Meteor Smart Package

[sendgrid](https://github.com/sendgrid/sendgrid-nodejs) NPM module packaged for Meteor.

### How to use?

1. Install [meteorite](https://github.com/oortcloud/meteorite)
2. `mrt add sendgrid`
3. Initialize with:

    var sendgrid = new Sendgrid({
        // options...
    });
